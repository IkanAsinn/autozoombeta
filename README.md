## Usage Instructions
1. Download the code
    ```
    git clone https://github.com/ikanasinn/autoZoom.git
    ```
    or

    Download the zip file [here](https://github.com/IkanAsinn/autozoom/archive/refs/heads/main.zip)

2. Run autofetch.exe (Login with your BinusMaya Account)
3. Run autoZoom.exe
4. Enjoy!
